## Getting started

1. Clone the repository

```sh
git clone git@github.com:OpenbookOrg/openbook-app.git
cd openbook-app
```

2. To verify flutter is installed and working.

````sh
flutter doctor
````

3. Open the iOS/Android simulator or connect a device.

4. Run the flutter app

````sh
flutter run
````
